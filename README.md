# hanze-multiomics-workshop
repository of code for the Hanze multi-omics course using cell type annotation

This repo contains two files:
'*hanze_multiomics_course.Rmd*' The R markdown script to create the tutorial page
'*hanze_multiomics_course.html*' The output tutorial webpage to display

The HTML can viewed here:

https://htmlpreview.github.io/?https://github.com/royoelen/hanze-multiomics-workshop/blob/master/hanze_multiomics_course.html
